# Tic Tac Toe

## Play Game

```
[$] git clone https://github.com/somtodev/tic-tac-toe-c
[$] cd tic-tac-toe-c
```
###### For Linux / Mac OS
```
[$] bash Run
```

###### For Windows
```
[$] start Run
```
---

### Requirement

> GCC Compiler
